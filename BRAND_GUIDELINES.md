# Manual de Marca CCI Group

## Identidad de Marca

**Nombre Oficial:** CCI Group

**Descripción:** Soluciones empresariales profesionales que transforman negocios con confianza, seriedad y dinamismo.

---

## Paleta de Colores Oficial

### Color Primario: Azul CCI

**Uso:** Texto del logo, elementos de marca serios, textos principales, fondos secundarios

**HEX:** `#001A70`

**RGB:** 0, 26, 112

**Significado:** Transmite seriedad, profesionalismo y confianza

**Aplicaciones:**
- Texto principal
- Elementos de marca institucional
- Fondos secundarios
- Iconos serios/importantes

---

### Color Secundario: Rojo CCI

**Uso:** Símbolo del logo, Llamadas a la Acción (CTAs), palabras clave destacadas

**HEX:** `#EE2A31`

**RGB:** 238, 42, 49

**Significado:** Aporta energía, dinamismo y un toque moderno

**Aplicaciones:**
- Botones principales (Contacto, Solicitar Cotización)
- Links y elementos interactivos
- Palabras clave destacadas
- Iconos de acción
- Hover states

## Logotipo

- **Color:** Azul CCI (#001A70) para texto + Rojo CCI (#EE2A31) para símbolo
- **Tamaño mínimo:** 50px de ancho
- **Espacio de respiro:** Margen mínimo de 10px alrededor del logo
- **Variantes:**
  - Logo completo (texto + símbolo)
  - Solo símbolo (para favicon, redes sociales)

## Uso de Colores en la Web

### Ejemplos de Aplicación

#### Botones Primarios (CTAs)
```
Background: #EE2A31 (Rojo CCI)
Text: Blanco (#FFFFFF)
Hover: #D41F26 (Rojo más oscuro)
```

#### Enlaces
```
Color: #EE2A31 (Rojo CCI)
Hover: #001A70 (Azul CCI)
Underline: Opcional
```

#### Fondos Secundarios
```
Background: #001A70 (Azul CCI)
Text: Blanco (#FFFFFF)
Para secciones importantes del sitio
```

#### Textos Destacados
```
Color: #EE2A31 (Rojo CCI) - Palabras clave
Color: #001A70 (Azul CCI) - Textos serios/formales
```

## Consistencia de Marca

- Usar SIEMPRE los códigos HEX exactos especificados
- No usar variantes de estos colores sin justificación
- Mantener coherencia en todos los canales digitales
- Revisar regularmente que el sitio cumpla con estas directrices
